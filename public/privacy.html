Privacy Policy

Effective date: March 14, 2026

KeyShortcut is built with privacy as a core principle. The app operates entirely offline and collects zero user data.

Data Collection

KeyShortcut does not collect, store, or transmit any personal data. There are no analytics frameworks, no tracking pixels, no third-party SDKs, and no advertising of any kind.

Network Access

KeyShortcut makes zero network requests. The app contains no networking code and does not communicate with any server. It works entirely offline.

Local Storage

KeyShortcut stores your preferences locally on your device using macOS UserDefaults within the app sandbox. This includes:

  • Favorited shortcuts
  • Custom shortcuts you create
  • Window position and appearance settings
  • Language preference
  • Imported shortcut pack file references

This data never leaves your Mac. Uninstalling KeyShortcut removes all associated data.

Accessibility Permissions

KeyShortcut may request macOS Accessibility permission for two optional features:

  • Follow Key Presses — monitors which keys you press to highlight matching shortcuts in real time. This data is processed in memory and never stored or transmitted.
  • Active App Detection — identifies the frontmost application to show relevant shortcuts. Only the app's bundle identifier is read; no window content is accessed.

Both features are disabled by default and require explicit opt-in.

Apple Diagnostics

KeyShortcut uses Apple's MetricKit framework, which allows Apple to collect anonymized crash reports and performance metrics through the standard macOS diagnostics pipeline. This data is processed by Apple, not by us. You can control this in System Settings > Privacy & Security > Analytics & Improvements.

Children's Privacy

KeyShortcut does not collect any data from any user, including children.
