# KeySticker Website

Product landing page for [KeySticker](https://keysticker.app) — a Mac menu bar app that shows keyboard shortcuts for any active application.

## Tech Stack

- **React 19** + **React Router v7** (framework mode, SSR + static pre-rendering)
- **Vite 7** build tooling
- **Tailwind CSS 4** styling
- **Cloudflare Pages** hosting

## Getting Started

```bash
pnpm install
pnpm dev
```

## Available Scripts

| Script | Description |
|--------|-------------|
| `pnpm dev` | Start dev server with HMR |
| `pnpm build` | Production build (SSR + pre-render) |
| `pnpm preview` | Preview production build locally |
| `pnpm lint` | Run ESLint |
| `pnpm test` | Run Vitest test suite |
| `pnpm run deploy` | Build + deploy to Cloudflare Pages |
